# 示例输入：女版

把下面这段话粘贴给 Claude Code（已装好 SKILL）或贴到任意 LLM（system prompt 装好 SKILL.md 后）。

---

```
月老看看：

女 28 岁，163cm 50kg，985 硕士，
杭州外企产品运营，年薪到手 35w，
存款 30w，杭州父母全款一套房可继承，无车，
父母 55 岁公务员（杭州本地，无负债，有姐姐已婚已育在杭州），
未婚，自评颜值 6.5 分。
```

---

## 预期输出形态

LLM 会输出两段：

**第一段 摘要（yaml）**
- basic_pan / verdict / upper / lower / plus / minus / actions

**第二段 详细（1200-1500 字）**
直播间口播长版画像 —— 532 打分、上下限展开、避雷清单、老御总结。

> 注意：SKILL 里有"称谓铁律"——客户是女就全程"姐妹"，不会出现"兄弟"称谓。
