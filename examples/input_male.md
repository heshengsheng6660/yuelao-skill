# 示例输入：男版

把下面这段话粘贴给 Claude Code（已装好 SKILL）或贴到任意 LLM（system prompt 装好 SKILL.md 后）。

---

```
月老看看：

男 30 岁，178cm 70kg，211 本科，
上海互联网中厂产品经理，年薪到手 60w，
存款 80w，上海一套房（购入 600w 贷款 300w 已还 50w），无车，
父母 60 岁退休（江苏二线，有养老金，无负债），独生子，
未婚，自评颜值 6 分。
```

---

## 预期输出形态

LLM 会输出两段：

**第一段 摘要（yaml）**
- basic_pan：一句话基本盘
- verdict：一句话戳破式总评
- upper / lower：上下限维度
- plus / minus：最大加分 / 最大命门
- actions：3 条可执行行动

**第二段 详细（1200-1500 字）**
直播间口播长版画像 —— 532 打分、上下限展开、避雷清单、老御总结。

> 注意：不同模型输出风格略有差异，Claude 系列最贴合鳌烨腔调。
