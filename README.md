### 问：男 28，180cm 65kg，985 本科，杭州大厂程序员 80w，存款 50w 无房无车，父母浙江县城退休有养老，独生子，自评 7 分单身。月老看看？

> **鳌烨**：兄弟我跟你说句实话——你这盘 28 岁 180 大厂 80w 单身，**你最大的问题不是基本盘，是你压根没用对地方**。
>
> 80w 年薪、5 年大厂、28 岁——这个段位的男生在杭州**早该上车了**。你存款 50w、无房无车，意味着这五年钱都吃喝玩乐花掉了。**这不是穷，这是规划差**。规划差的男生，姐妹们一眼能看出来。
>
> 自评 7 分？兄弟，**送过你花、写过情书、托闺蜜要过微信的姐妹有几个**？如果没有，那你大概率是 5.5 分。**自评 7 分大概率虚高 1.5 分**，这是黄金法则。
>
> 你的上限：杭州本地 5 分以上独生女、24-27 岁、外企/国企/互联网中厂；下限：杭州非本地 985 硕、26-30 岁、4.5 分。**你不缺收入，缺的是会撩 + 上车**。
>
> 6 个月内做两件事：① 上车一套房（哪怕首付），把"无房"标签摘掉；② 学撩——线下圈子、表达课、形象升级，**不要准备完美才行动**。

> 🎯 运用了 **532 打分体系**（80w 物质拉满但情绪/性格 0 分）、**自评颜值反推**（自评 7 实际 5.5 法则）、**上车标签**（28 岁无房的婚恋市场硬伤）、**"能开口就不会单身"反推**（80w 还单身 = 不是钱的问题）、**杭州黄金期窗口**（30 岁前必须上车）

## 在自己电脑上跑

**自己跑 = 数据不上传任何服务器 + 可以 fork 改造**。

### 方式 1：Claude Code（效果最好，原生 skill 格式）

```bash
git clone https://github.com/heshengsheng6660/yuelao-skill.git
mkdir -p ~/.claude/skills/yuelao
cp yuelao-skill/SKILL.md ~/.claude/skills/yuelao/

# 打开 Claude Code，输入基本盘
# 例："男 30 岁 178 上海互联网 60w 月老看看"
# Claude 自动触发 yuelao skill
```

### 方式 2：DeepSeek / GPT / Cursor / 任意 LLM

把 `SKILL.md` 整段贴到 system prompt，然后发用户基本盘即可。

```python
from openai import OpenAI
client = OpenAI(api_key="...", base_url="https://api.deepseek.com")

with open("SKILL.md", "r") as f:
    system_prompt = f.read()

response = client.chat.completions.create(
    model="deepseek-chat",
    messages=[
        {"role": "system", "content": system_prompt},
        {"role": "user", "content": "男 30 岁 178 上海互联网 60w 月老看看"}
    ]
)
print(response.choices[0].message.content)
```

**模型推荐**：
- 🥇 **Claude Sonnet / Opus** — 最贴鳌烨腔调，长上下文友好
- 🥈 **GPT-4 / GPT-4o** — 结构稳，腔调略平
- 🥉 **DeepSeek-V3** — 中文表达强，性价比最高
- ⚠️ **Qwen / 文心 / 豆包** — 能跑但腔调差异较大，需要微调 prompt
